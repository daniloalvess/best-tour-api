# Best Tour API

API para encontrar o melhor trajeto, com base nas coordenadas informadas.

## Exemplo de requisição
```json
{
  "locations": [
    {
      "lat": -18.8785153,
      "lon": -41.9698963
    },
    {
      "lat": -18.510516,
      "lon": -41.866565
    },
    {
      "lat": -17.630215,
      "lon": -41.6182573
    },
    {
      "lat": -18.8917509,
      "lon": -41.9637153
    }
  ]
}
```
